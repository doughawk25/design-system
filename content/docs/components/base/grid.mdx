Lay out content in a grid with optional row and column guides. Use `Grid` and `GridCell` to build layouts that match the same conceptual patterns as the Geist grid: plain grid, basic cells, solid cells that occlude guides, responsive layouts, hidden row or column guides, overlaying cells, and guide clipping on specific cells.

## Usage

```tsx showLineNumbers
import { Grid, GridCell } from "@/components/ui/grid"
```

```tsx showLineNumbers
<Grid columns={3} rows={2} gap={8}>
  <GridCell>1</GridCell>
  <GridCell solid>2</GridCell>
  <GridCell column={1} row={2} colSpan={2} clipGuides>3</GridCell>
</Grid>
```

## API

- **Grid**: `columns`, `rows` (number or CSS template string), `gap`, `showRowGuides`, `showColumnGuides`, `minHeight`.
- **GridCell**: `column`, `row`, `colSpan`, `rowSpan`, `solid`, `overlay`, `clipGuides`.
